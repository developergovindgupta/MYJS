String.toCodeArray()

Target:

Applicable with String Data Type.

Description :

return array of character Codes of string's every characters.


Syntax:

String.toCodeArray()

Usase:

  1. "Hello World".toCodeArray()
    It return Array [72, 101, 108, 108, 111, 32, 87, 111, 114, 108, 100]
  2. "MyJs : Make your javascript coding easy.".toCodeArray()
    It return Array [77, 121, 74, 115, 32, 58, 32, 77, 97, 107, 101, 32, 121, 111, 117, 114, 32, 106, 97, 118, 97, 115, 99, 114, 105, 112, 116, 32, 99, 111, 100, 105, 110, 103, 32, 101, 97, 115, 121, 46]
  3. "MyJs : Make your javascript coding easy.".toCodeArray().distinct()
    It return Array [77, 121, 74, 115, 32, 58, 97, 107, 101, 111, 117, 114, 106, 118, 99, 105, 112, 116, 100, 110, 103, 46]
  4. "This library add many dom prototype extension methods to make javascript coding easy.".toCodeArray().forEach(x=>console.log(x));

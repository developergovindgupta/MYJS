Date.addDays(Number)

Target:

Applicable with Date Data Type.

Description :

Add/Subtract days from Date and return new Date object.


Syntax:

Date.addDays(number)

Usase:

var dd = '1 Apr 2020'.toDateTime();
  1. dd.addDays(5);
    It return "Mon Apr 06 2020 00:00:00 GMT+0530 (India Standard Time)"
  2. dd.addDays(-5);
    It return "Fri Mar 27 2020 00:00:00 GMT+0530 (India Standard Time)"

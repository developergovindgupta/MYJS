String.toCamelCase()

Target:

Applicable with String Data Type.

Description :

convert string to a single word in camel case format. Every words first letter in capital letter except first word.


Note: convert into camelCase from PascalCase, snake_case and kebab-case.


Syntax:

String.toCamelCase()

Usase:

  1. " hello     how     are     you ? ".toCamelCase()
    It return string "helloHowAreYou?"
  2. "hello-how-are-you-?".toCamelCase()
    It return string "helloHowAreYou?"
  3. "hello_how_are_you_?".toCamelCase()
    It return string "helloHowAreYou?"
  4. "HelloHowAreYou?".toCamelCase()
    It return string "helloHowAreYou?"

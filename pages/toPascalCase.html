String.toPascalCase()

Target:

Applicable with String Data Type.

Description :

convert string to a single word in camel case format. Every words first letter in capital letter.


Note: convert into PascalCase from camelCase, snake_case and kebab-case.


Syntax:

String.toPascalCase()

Usase:

  1. " hello     how     are     you ? ".toPascalCase()
    It return string "HelloHowAreYou?"
  2. "hello-how-are-you-?".toPascalCase()
    It return string "HelloHowAreYou?"
  3. "hello_how_are_you_?".toPascalCase()
    It return string "HelloHowAreYou?"
  4. "helloHowAreYou?".toPascalCase()
    It return string "HelloHowAreYou?"

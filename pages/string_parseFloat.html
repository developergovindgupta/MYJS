String.parseFloat([nanValue])

Target:

Applicable with String Data Type.

Description :

convert string to floating point number.


Syntax:

String.parseFloat([nanValue])

Note : string is Not a Number then it return NaN or passed value (nanValue)

Usase:

  1. "55.25".parseFloat()
    It return number 55.25
  2. "132abcd".parseFloat()
    It return number 123
  3. "1,23,456.789".parseFloat()
    It return number 1
  4. "abcd132".parseFloat()
    It return number NaN
  5. "abcd132".parseFloat(0)
    It return number 0
  6. "abcd132".parseFloat(-1)
    It return number -1

String.toDateTime()

Target:

Applicable with String Data Type.

Description :

convert string to Date object. Date should be "dd-MMM-yyyy hh:mm:ss tt" or "dd-MM-yyyy HH:mm:ss"


Note: if date not passed then current date and given time will return.
In case of invalid date string it return "1-Jan-1900 00:00 AM"

Syntax:

String.toDateTime()

Usase:

  1. "11-Oct-2019".toDateTime()
    It return Date "Fri Oct 11 2019 00:00:00 GMT+0530 (India Standard Time)"
  2. "11/Oct/2019".toDateTime()
    It return Date "Fri Oct 11 2019 00:00:00 GMT+0530 (India Standard Time)"
  3. "11.Oct.2019".toDateTime()
    It return Date "Fri Oct 11 2019 00:00:00 GMT+0530 (India Standard Time)"
  4. "5Jun2019".toDateTime()
    It return Date "Wed Jun 05 2019 00:00:00 GMT+0530 (India Standard Time)"
  5. "15Jun".toDateTime()
    It return Date "Sat Jun 15 2019 00:00:00 GMT+0530 (India Standard Time)"
  6. "28, Jun".toDateTime()
    It return Date "Fri Jun 28 2019 00:00:00 GMT+0530 (India Standard Time)"
  7. "11, oct - 2019".toDateTime()
    It return Date "Fri Oct 11 2019 00:00:00 GMT+0530 (India Standard Time)"
  8. "July,2019".toDateTime()
    It return Date "Mon Jul 01 2019 00:00:00 GMT+0530 (India Standard Time)"
  9. "July 19".toDateTime()
    It return Date "Mon Jul 01 2019 00:00:00 GMT+0530 (India Standard Time)"
  10. "05/07/2019".toDateTime()
    It return Date "Fri Jul 05 2019 00:00:00 GMT+0530 (India Standard Time)"
  11. "5/12/2019".toDateTime()
    It return Date "Thu Dec 05 2019 00:00:00 GMT+0530 (India Standard Time)"
  12. "5-6-19".toDateTime()
    It return Date "Wed Jun 05 2019 00:00:00 GMT+0530 (India Standard Time)"
  13. "5-2019".toDateTime()
    It return Date "Wed May 01 2019 00:00:00 GMT+0530 (India Standard Time)"
  14. "7-82".toDateTime()
    It return Date "Thu Jul 01 1982 00:00:00 GMT+0530 (India Standard Time)"
  15. "05-Oct-2019 11:30:45 PM".toDateTime()
    It return Date "Sat Oct 05 2019 23:30:45 GMT+0530 (India Standard Time)"
  16. "11:30:45 PM".toDateTime()
    It return Date "Current Date 23:30:45 GMT+0530 (India Standard Time)"
  17. "21:30:45".toDateTime()
    It return Date "Current Date 21:30:45 GMT+0530 (India Standard Time)"
  18. "8:30 AM".toDateTime()
    It return Date "Current Date 08:30:00 GMT+0530 (India Standard Time)"

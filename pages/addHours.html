Date.addHours(Number)

Target:

Applicable with Date Data Type.

Description :

Add/Subtract hours from Date and return new Date object.


Syntax:

Date.addHours(number)

Usase:

var dd = '1 Apr 2020 12:00 AM'.toDateTime();
  1. dd.addHours(1);
    It return "Wed Apr 01 2020 01:00:00 GMT+0530 (India Standard Time)"
  2. dd.addHours(-1);
    It return "Tue Mar 31 2020 23:00:00 GMT+0530 (India Standard Time)"

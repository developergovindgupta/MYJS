String.reverse()

Target:

Applicable with String Data Type.

Description :

Reverses the string letter by letter right to left.


Syntax:

String.reverse()

Usase:

  1. " hello     how     are     you ? ".reverse()
    It return string " ? uoy     era     woh     olleh "
  2. "hello-how-are-you-?".reverse()
    It return string "?-uoy-era-woh-olleh"

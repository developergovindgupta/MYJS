String.toKebabCase()

Target:

Applicable with String Data Type.

Description :

convert string to a single word in kebab-case format. Every words separated by '-'.


Note: convert into kebab-case from PascalCase, snake_case and camel-case.


Syntax:

String.toKebabCase()

Usase:

  1. " hello     how     are     you ? ".toKebabCase()
    It return string "hello-how-are-you-?"
  2. "helloHowAreYou?".toKebabCase()
    It return string "hello-how-are-you-?"
  3. "hello_how_are_you_?".toKebabCase()
    It return string "hello-how-are-you-?"
  4. "HelloHowAreYou?".toKebabCase()
    It return string "hello-how-are-you-?"

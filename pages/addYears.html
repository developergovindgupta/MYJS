Date.addYears(Number)

Target:

Applicable with Date Data Type.

Description :

Add/Subtract years from Date and return new Date object.


Syntax:

Date.addYears(number)

Usase:

var dd = '1 Apr 2020'.toDateTime();
  1. dd.addYears(1);
    It return "Thu Apr 01 2021 00:00:00 GMT+0530 (India Standard Time)"
  2. dd.addYears(-1);
    It return "Mon Apr 01 2019 00:00:00 GMT+0530 (India Standard Time)"

String.toNumber([nanValue])

Target:

Applicable with String Data Type.

Description :

convert string to number.


Syntax:

String.toNumber([nanValue])

Note : string is NaN then it return (0) or passed value (nanValue)

Usase:

  1. "55.25".toNumber()
    It return number 55.25
  2. "10,500.50".toNumber()
    It return number 10500.5
  3. "1,23,456.789".toNumber()
    It return number 123456.789
  4. "132abcd".toNumber()
    It return number 123
  5. "abcd132".toNumber()
    It return number 0
  6. "abcd132".toNumber(-1)
    It return number -1

String.toArray()

Target:

Applicable with String Data Type.

Description :

It convert string to characters array.


Syntax:

String.toArray()

Usase:

  1. "Hello World".toArray()
    It return Array ["H", "e", "l", "l", "o", " ", "W", "o", "r", "l", "d"]
  2. "MyJs : Make your javascript coding easy.".toArray()
    It return Array ["M", "y", "J", "s", " ", ":", " ", "M", "a", "k", "e", " ", "y", "o", "u", "r", " ", "j", "a", "v", "a", "s", "c", "r", "i", "p", "t", " ", "c", "o", "d", "i", "n", "g", " ", "e", "a", "s", "y", "."]
  3. "MyJs : Make your javascript coding easy.".toArray().distinct()
    It return Array ["M", "y", "J", "s", " ", ":", "a", "k", "e", "o", "u", "r", "j", "v", "c", "i", "p", "t", "d", "n", "g", "."]
  4. "This library add many dom prototype extension methods to make javascript coding easy.".toArray().forEach(x=>console.log(x));
    It return in console

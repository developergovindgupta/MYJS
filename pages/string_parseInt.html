String.parseInt([nanValue])

Target:

Applicable with String Data Type.

Description :

convert string to number.


Syntax:

String.parseInt([nanValue])

Note : string is Not a Number then it return NaN or passed value (nanValue)

Usase:

  1. "55.25".parseInt()
    It return number 55
  2. "132abcd".parseInt()
    It return number 123
  3. "1,23,456.789".parseInt()
    It return number 1
  4. "abcd132".parseInt()
    It return number NaN
  5. "abcd132".parseInt(0)
    It return number 0
  6. "abcd132".parseInt(-1)
    It return number -1

Date.addMinutes(Number)

Target:

Applicable with Date Data Type.

Description :

Add/Subtract Minutes from Date and return new Date object.


Syntax:

Date.addMinutes(number)

Usase:

var dd = '1 Apr 2020 12:00 AM'.toDateTime();
  1. dd.addMinutes(1);
    It return "Wed Apr 01 2020 00:01:00 GMT+0530 (India Standard Time)"
  2. dd.addMinutes(-1);
    It return "Tue Mar 31 2020 23:59:00 GMT+0530 (India Standard Time)"

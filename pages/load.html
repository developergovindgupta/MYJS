DOM.load("url")

Target:

Applicable with any DOM object having innerHTML attribute.

Description :

It load passed page url content in target DOM element asynchronously. It follows Promise structure.

Syntax:

DOMElement.load('url').then(function).catch(function)

Usase:

  1. document.getElementById('divTarget').load('page.html')
  2. document.querySelector('#divTarget').load('page.html')
  3. document.getElementById('divTarget').load('page.html').then(function(obj){
    • //After Load Success Process
      //obj target element as 'divTarget' in this example.
    }).catch(function(ex){
    • //After Load Fail Process
    })

String.toSnakeCase()

Target:

Applicable with String Data Type.

Description :

convert string to a single word in snake_case format. Every words separated by '_'.


Note: convert into snake_case from PascalCase, camelCase and kebab-case.


Syntax:

String.toSnakeCase()

Usase:

  1. " hello     how     are     you ? ".toSnakeCase()
    It return string "hello_how_are_you_?"
  2. "hello-how-are-you-?".toSnakeCase()
    It return string "hello_how_are_you_?"
  3. "helloHowAreYou?".toSnakeCase()
    It return string "hello_how_are_you_?"
  4. "HelloHowAreYou?".toSnakeCase()
    It return string "hello_how_are_you_?"
